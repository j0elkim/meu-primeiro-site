# Meu Primeiro Site

Página web simples criada como meu primeiro projeto versionado com Git e GitHub.

## O que é

Um cartão de apresentação com HTML, CSS e um pouco de JavaScript.

## Como abrir

Abra o arquivo `index.html` no navegador (clique duas vezes nele).

## Arquivos

- `index.html` — a estrutura da página
- `style.css` — o estilo (cores, layout)
- `script.js` — a interatividade do botão
